# Установка SDK

Inngest SDK позволяет постепенно добавлять надёжные, устойчивые функции в существующие проекты. Функции могут автоматически запускаться событиями или выполняться по расписанию без какой-либо инфраструктуры — как полностью бессерверные, так и в рамках существующего HTTP-сервера.

- Работает с любым фреймворком и платформой через HTTP-вызовы функций
- Поддерживает бессерверных провайдеров без дополнительной инфраструктуры
- Полностью поддерживает TypeScript из коробки
- Позволяет локально тестировать код без дополнительной настройки



## Начало работы

<LanguageSelector>

<LanguageSection show="typescript">

### Установка

Для начала установите SDK с помощью предпочитаемого менеджера пакетов:

<CodeGroup>
```shell {{ title: "npm" }}
npm install inngest
```
```shell {{ title: "yarn" }}
yarn add inngest
```
```shell {{ title: "pnpm" }}
pnpm add inngest
```
```shell {{ title: "bun" }}
bun add inngest
```
</CodeGroup>

### Настройка

После установки Inngest создайте клиент Inngest, который в дальнейшем будет использоваться для определения функций и отправки событий:

```typescript
import { Inngest } from "inngest";

export const inngest = new Inngest({
  id: "my-app",
});
```


</LanguageSection>

<LanguageSection show="go">


<LanguageSection show="python">

### Установка



Для начала установите SDK с помощью `pip`:



```shell
pip install inngest
```

### Настройка

После установки Inngest создайте клиент Inngest, который в дальнейшем будет использоваться для определения функций и отправки событий:

```python
import inngest
import logging

# Создаём клиент Inngest
inngest_client = inngest.Inngest(
    app_id="app_example",
    logger=logging.getLogger("uvicorn"),
)
```
