# Промежуточное ПО Sentry


Использование промежуточного ПО Sentry полезно для:
- Захвата исключений для отчётности
- Добавления трассировки к каждому запуску функции
- Включения полезного контекста для каждого исключения и трассировки, например ID функции и имён событий


## Установка

<LanguageSelector>


  <LanguageSection show="python">

    `SentryMiddleware` поставляется как часть пакета `inngest`:

    ```py
    import inngest
    from inngest.experimental.sentry_middleware import SentryMiddleware

    import sentry_sdk

    # Initialize Sentry as usual wherever is appropriate
    sentry_sdk.init(
        traces_sample_rate=1.0,
        profiles_sample_rate=1.0,
    )

    inngest_client = inngest.Inngest(
        app_id="my-app",
        middleware=[SentryMiddleware],
    )
    ```


  </LanguageSection>

  <LanguageSection show="typescript">

    Установите [пакет `@inngest/middleware-sentry`](https://www.npmjs.com/package/@inngest/middleware-sentry) и настройте его следующим образом:

    ```ts
    import * as Sentry from "@sentry/node";
    import { Inngest } from "inngest";
    import { sentryMiddleware } from "@inngest/middleware-sentry";

    // Initialize Sentry as usual wherever is appropriate
    Sentry.init(...);

    const inngest = new Inngest({
      id: "my-app",
      middleware: [sentryMiddleware()],
    });
    ```

  <Callout>
    Требует `inngest@>=4.0.0` и `@sentry/*@>=8.0.0`.
  </Callout>

  </LanguageSection>


</LanguageSelector>
