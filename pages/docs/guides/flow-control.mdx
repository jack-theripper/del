# Управление потоком (Flow Control)

Управление потоком — это критически важная часть создания надежных приложений. Оно позволяет управлять потоком данных и событий в вашем приложении, что помогает эффективно распределять ресурсы, предотвращать перегрузку систем и гарантировать отзывчивость и надежность вашего приложения.

Существует несколько методов управления потоком для каждой функции Inngest. Узнайте о каждом методе и о том, как использовать их в ваших функциях:

  </Card>
  <Card title="Дросселирование (Throttling)" icon={<RiSlowDownFill className="text-basis h-4 w-4" />} href={'/docs/guides/throttling'}>
    Ограничьте пропускную способность выполнения функции за определенный период времени. Идеально подходит для соблюдения лимитов сторонних API.
  </Card>

  </Card>

  </Card>
  <Card title="Приоритет (Priority)" icon={<RiGitPullRequestFill className="rotate-90 text-basis h-4 w-4" />} href={'/docs/guides/priority'}>
