# Массовая отмена

{/* TODO - Link the sleeps and waits to guides when we move those from references to guides */}
В Inngest ваши функции могут выполняться или находиться в режиме паузы в течение длительного времени. У вас может быть функция с сотнями шагов, или вы можете использовать [`step.sleep`](/docs/reference/functions/step-sleep), [`step.sleepUntil`](/docs/reference/functions/step-sleep-until) или [`step.waitForEvent`](/docs/reference/functions/step-wait-for-event). Иногда в вашей системе происходят события, из-за которых завершение выполнения функции становится ненужным — в таких случаях требуется отмена.

Inngest предоставляет как API для массовой отмены, так и соответствующий интерфейс в панели управления. API для массовой отмены предлагает большую гибкость благодаря поддержке сопоставления выражений событий, в то время как [интерфейс массовой отмены](/docs/platform/manage/bulk-cancellation), доступный на платформе, обеспечивает быстрый способ отмены нежелательных запусков функций.

## Массовая отмена через REST API

Вы также можете массово отменять функции через [REST API](https://api-docs.inngest.com/docs/inngest-api). Это полезно, если вам нужно отменить большое количество функций в определенном диапазоне.

С помощью эндпоинта `POST /cancellations` вы можете отменять функции, указав `app_id`, `function_id`, а также временной диапазон через метки `started_after` и `started_before`. Вы также можете при желании указать условие `if`, чтобы отменить только те функции, которые соответствуют [заданному выражению](/docs/guides/writing-expressions).

<CodeGroup tag="POST" label="api.inngest.com/v1/cancellations">

```bash {{ title: 'cURL' }}
curl -X POST https://api.inngest.com/v1/cancellations \
  -H 'Authorization: Bearer signkey-prod-<YOUR-SIGNING-KEY>' \
  -H 'Content-Type: application/json' \
  --data '{
    "app_id": "acme-app",
    "function_id": "schedule-reminder",
    "started_after": "2024-01-21T18:23:12.000Z",
    "started_before": "2024-01-22T14:22:42.130Z",
    "if": "event.data.userId == 'user_o9235hf84hf'"
  }'
```

</CodeGroup>

При успешном выполнении будет возвращен ответ с ID отмены и данными задачи на отмену:

```json {{ title: 'Response' }}
{
  "id": "01HMRMPE5ZQ4AMNJ3S2N79QGRZ",
  "environment_id": "e03843e1-d2df-419e-9b7b-678b03f7398f",
  "function_id": "schedule-reminder",
  "started_after": "2024-01-21T18:23:12.000Z",
  "started_before": "2024-01-22T14:22:42.130Z",
  "if": "event.data.userId == 'user_o9235hf84hf'"
}
```

Чтобы узнать больше, прочтите полный [справочник по REST API](https://api-docs.inngest.com/docs/inngest-api).
