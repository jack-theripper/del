# `inngest/function.failed` {{ className: "not-prose" }}

Событие `inngest/function.failed` отправляется каждый раз, когда какая-либо функция завершается с ошибкой в вашем [окружении Inngest](/docs/platform/environments).

Это событие можно использовать для централизованного отслеживания всех сбоев функций, что позволяет отправлять метрики, оповещения или события во [внешние системы, такие как Datadog или Sentry](/docs/examples/track-failures-in-datadog), для всех ваших функций Inngest.

Наши SDK предлагают сокращённые варианты обработчика [«при сбое»](#related-resources), которые можно использовать для обработки этого события для конкретной функции.
