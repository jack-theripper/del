# `inngest/function.cancelled` {{ className: "not-prose" }}

Событие `inngest/function.cancelled` отправляется каждый раз, когда отменяется отдельная функция в вашем [окружении Inngest](/docs/platform/environments). Событие будет отправлено, если функция отменена через [событие `cancelOn`](/docs/features/inngest-functions/cancellation/cancel-on-events), [таймауты функции](/docs/features/inngest-functions/cancellation/cancel-on-timeouts), [REST API](/docs/guides/cancel-running-functions) или [массовую отмену](/docs/platform/manage/bulk-cancellation).

Это событие можно использовать для выполнения очистки или аналогичных действий для конкретной функции, а также для отслеживания отмен функций во внешних системах, таких как Datadog.

<Callout>
  Вы можете написать функцию, которая использует событие `"inngest/function.cancelled"` с необязательным параметром `if` для фильтрации и обработки конкретной функции по `function_id`.
</Callout>
