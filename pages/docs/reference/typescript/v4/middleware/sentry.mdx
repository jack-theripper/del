# Промежуточное ПО Sentry

Промежуточное ПО Sentry перехватывает исключения, добавляет трассировку и включает полезный контекст (ID функции, имена событий) для каждого запуска функции.

Смотрите [руководство по промежуточному ПО Sentry](/docs/features/middleware/sentry-middleware) для инструкций по настройке и использованию.
