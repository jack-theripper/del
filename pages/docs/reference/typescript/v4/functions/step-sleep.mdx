# Sleep `step.sleep()`


Используйте `step.sleep()` для приостановки выполнения функции на заданную длительность. Это полезно для добавления задержек между шагами, например при ожидании перед отправкой повторного письма или опросе внешнего сервиса с интервалами.

## `step.sleep(id, duration): Promise`

<Row>
  <Col>
    <Properties>
      <Property name="id" type="string" required>
        ID шага. Будет отображаться в логах функции и используется для мемоизации состояния шага между версиями функции.
      </Property>
      <Property name="duration" type="number | string | Temporal.Duration" required>
        Длительность паузы:

        * `number` — миллисекунды
        * `string` — совместимая с пакетом [ms](https://npm.im/ms) строка, например `"30m"`, `"3 hours"` или `"2.5d"`
        * [`Temporal.Duration`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Temporal/Duration)
      </Property>
    </Properties>
  </Col>
  <Col>
  ```ts
  // Спать 30 минут
  const thirtyMins = Temporal.Duration.from({ minutes: 30 });
  await step.sleep("wait-with-temporal", thirtyMins);

  await step.sleep("wait-with-string", "30m");
  await step.sleep("wait-with-string-alt", "30 minutes");
  await step.sleep("wait-with-ms", 30 * 60 * 1000);
  ```
  </Col>
</Row>

<Callout>
  `step.sleep()` должен вызываться с `await` или другим обработчиком Promise для корректной работы функции.
</Callout>
