# Sleep until `step.sleepUntil()`


Используйте `step.sleepUntil()` для приостановки выполнения функции до конкретной даты и времени. Это полезно, когда нужно ждать до известного момента времени, например до окончания пробного периода или запланированного дедлайна.

## `step.sleepUntil(id, datetime): Promise`

<Row>
  <Col>
    <Properties>
      <Property name="id" type="string" required>
        ID шага. Будет отображаться в логах функции и используется для мемоизации состояния шага между версиями функции.
      </Property>
      <Property name="datetime" type="Date | string | Temporal.Instant | Temporal.ZonedDateTime" required>
        Дата и время для возобновления выполнения функции. Может быть:

        * Объектом [`Date`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date)
        * Строкой даты и времени в [формате, принимаемом объектом `Date`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date#date_time_string_format): `YYYY-MM-DDTHH:mm:ss.sssZ` или упрощённые формы `YYYY-MM-DD` или `YYYY-MM-DDHH:mm:ss`
        * [`Temporal.Instant`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Temporal/Instant)
        * [`Temporal.ZonedDateTime`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Temporal/ZonedDateTime)
      </Property>
    </Properties>
  </Col>
  <Col>
  ```ts
  // Спать до Нового года
  await step.sleepUntil("happy-new-year", "2024-01-01");

  // Спать до конца сентября
  await step.sleepUntil("wake-me-up", "2023-09-30T11:59:59");

  // Спать до конца текущей недели
  const date = dayjs().endOf("week").toDate();
  await step.sleepUntil("wait-for-end-of-the-week", date);

  // Спать до времени чая в Лондоне
  const teaTime = Temporal.ZonedDateTime.from("2025-05-01T16:00:00+01:00[Europe/London]");
  await step.sleepUntil("british-tea-time", teaTime);

  // Спать до конца дня
  const now = Temporal.Now.instant();
  const endOfDay = now.round({ smallestUnit: "day", roundingMode: "ceil" });
  await step.sleepUntil("done-for-today", endOfDay);
  ```
  </Col>
</Row>

<Callout>
  `step.sleepUntil()` должен вызываться с `await` или другим обработчиком Promise для корректной работы функции.
</Callout>
