# Workflow Kit
Workflow Kit позволяет создавать [пользовательские рабочие процессы (user-defined workflows)](/docs/guides/user-defined-workflows) с помощью Inngest, предоставляя набор действий рабочего процесса для **[Workflow Engine](/docs/reference/workflow-kit/engine)** и используя **[готовые React-компоненты](/docs/reference/workflow-kit/components-api)** для создания интерфейса редактора рабочих процессов.

## Установка


<CodeGroup>
```shell {{ title: "npm" }}
npm install @inngest/workflow-kit inngest
```
```shell {{ title: "pnpm" }}
pnpm add @inngest/workflow-kit inngest
```
```shell {{ title: "yarn" }}
yarn add @inngest/workflow-kit inngest
```
</CodeGroup>

<Callout>

**Предварительные условия**

Workflow Kit интегрируется с нашим [TypeScript SDK](/docs/reference/typescript).

Для его использования вам понадобится приложение с [настроенным Inngest](/docs/sdk/overview), готовое к [обработке функций Inngest](/docs/learn/serving-inngest-functions).

</Callout>
