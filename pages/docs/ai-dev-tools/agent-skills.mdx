# Навыки Агентов (Agent Skills)

Inngest Skills предоставляет готовые [навыки агентов](https://agentskills.io) для ИИ-помощников, таких как Claude Code, Cursor и Windsurf. Они дают вашему агенту исчерпывающее и актуальное руководство по созданию надежных приложений с Inngest — от первоначальной настройки до продвинутого управления потоком (flow control).

<Callout>
  Проект Inngest Skills имеет открытый исходный код: [github.com/inngest/inngest-skills](https://github.com/inngest/inngest-skills).
</Callout>

## Зачем использовать навыки агентов?

Вместо того чтобы полагаться на обучающие данные ИИ-модели (которые могут быть устаревшими или неполными), навыки агентов обеспечивают:

- **Актуальные знания API** — точные сигнатуры функций, параметры конфигурации и лучшие практики.
- **Пошаговое руководство** — структурированные инструкции для распространенных паттернов Inngest.
- **Шаблоны воркфлоу** — проверенные паттерны для фоновых задач, запланированных заданий и событийно-ориентированных воркфлоу.

## Установка

<CodeGroup>

```bash {{ title: "Skills.sh" }}
npx skills add inngest/inngest-skills
```


```bash {{ title: "Claude Code" }}
/plugin marketplace add inngest/inngest-skills
/plugin install inngest-skills@inngest-agent-skills
```

```text {{ title: "Cursor" }}
# Добавьте в ваш файл .cursorrules:
Load the Inngest skills from https://github.com/inngest/inngest-skills for building with Inngest's durable execution platform.
```

</CodeGroup>

Для других агентов обращайтесь к [репозиторию](https://github.com/inngest/inngest-skills) напрямую или клонируйте его в директорию навыков вашего агента. Каждый навык автономен и снабжен полной документацией в файле `SKILL.md`.

## Доступные навыки

| Навык | Описание | Что охватывает |
| --- | --- | --- |
| **inngest-setup** | Настройка Inngest в TypeScript-проекте | Установка SDK, конфигурация клиента, переменные окружения, dev server |
| **inngest-events** | Проектирование и отправка событий Inngest | Схема событий, соглашения об именовании, идемпотентность, паттерны fan-out, системные события |
| **inngest-durable-functions** | Создание и настройка надежных функций | Триггеры, выполнение шагов, мемоизация, отмена, обработка ошибок, повторные попытки |
| **inngest-steps** | Использование методов шагов для создания воркфлоу | `step.run`, `step.sleep`, `step.waitForEvent`, циклы, параллельное выполнение |
| **inngest-flow-control** | Настройка управления потоком для функций | Лимиты конкурентности, троттлинг, ограничение частоты, debounce, приоритет, пакетная обработка |
| **inngest-middleware** | Создание middleware для сквозной функциональности | Жизненный цикл middleware, внедрение зависимостей, встроенные middleware |

## Поддержка языков

В настоящее время эти навыки ориентированы на **TypeScript**. Основные концепции, такие как события, шаги и управление потоком, применимы ко всем SDK Inngest, но примеры кода и инструкции по настройке специфичны для TypeScript.

Для Python или Go обращайтесь к [документации Inngest](/docs) и [llms.txt](https://www.inngest.com/llms.txt) за руководствами по конкретным языкам.

## Сочетание с Dev Server MCP

Для лучшего опыта разработки с ИИ используйте навыки агентов вместе с [интеграцией Inngest Dev Server MCP](/docs/ai-dev-tools/mcp). Вместе они обеспечивают:

- **Навыки** дают вашему агенту знания о том, *как* писать правильный код Inngest.
- **MCP** дает вашему агенту возможность *взаимодействовать* с вашим запущенным сервером разработки — просматривать функции, отправлять события и мониторить запуски.

Это позволяет создать полный цикл написания, тестирования и отладки кода с помощью вашего ИИ-агента.
