# Документация Inngest

Inngest — это платформа для надежного выполнения кода (durable execution) на основе событий, которая позволяет запускать быстрый и отказоустойчивый код на любой платформе без управления очередями, инфраструктурой или состоянием.

Пишите функции на TypeScript, Python или Go для фоновых задач и задач по расписанию со встроенными шагами (steps). Мы берем на себя бэкенд-инфраструктуру, очереди, масштабирование, конкурентность, троттлинг, ограничение частоты запросов и наблюдаемость (observability).

## Документация для LLM

Версия документации Inngest, удобная для LLM, доступна в двух форматах: `llms.txt` и `llms-full.txt`. Они полезны для передачи контекста LLM, ИИ-IDE или аналогичным инструментам для ответов на вопросы об Inngest.

* [inngest.com/llms.txt](https://www.inngest.com/llms.txt) — оглавление документации, идеально подходит для небольших окон контекста или инструментов, которые могут сканировать документацию.
* [inngest.com/llms-full.txt](https://www.inngest.com/llms-full.txt) — вся документация в формате markdown.
